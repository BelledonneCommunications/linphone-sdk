            As of now we don't have any light weight unit testing framework for C as it is for C++ (CppUnit) or Java (JUnit). This intiative is taken to provide such a platform independent/version independent core framework and the OS/Environment specific wrappers around them. The wrappers will be finally provided for the following environments,
  • Console (Using standard C library)
  • Curses (On Unix using ncurses library)
  • Windows Based (Dialog based on Win9x/NT/2000/XWindows)
